## üêõ Troubleshooting

### Installation Issues
```bash
# If install_rag_deps.bat fails:
venv\Scripts\activate
pip install chromadb>=0.4.0 aiohttp>=3.9.0
```

### Apollo API Issues
- Check API key in Settings
- Verify credit balance
- Check rate limits (10 requests/second)

### No Results
- Select more industries (increase pool)
- Lower min_validation_score to 3/6
- Increase company size range
- Check Apollo search parameters

### Slow Generation
- RAG should be enabled (default)
- Check console for errors
- Verify chromadb installed correctly
